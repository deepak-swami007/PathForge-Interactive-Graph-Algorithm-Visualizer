# GraphX Rescue

GraphX Rescue is a 2D graph algorithm visualizer with a disaster-response theme.

The goal is to build a strong DSA project with the simplest web-deployable stack:

```text
HTML + CSS + Vanilla JavaScript
```

## Planned Features

- Rescue grid with start, target, blocked cells, bombs, and weighted zones.
- Pathfinding algorithms such as BFS, DFS, Dijkstra, and A*.
- Graph algorithms such as Bellman-Ford, Floyd-Warshall, Prim, Kruskal, Kosaraju, and Ford-Fulkerson.
- Visual stats for visited nodes, path length, risk cost, and runtime.
- Free deployment through GitHub Pages.

## Run Locally

Open `index.html` directly in a browser.

No installation is required.
